# Smart Geofence Controller

A Python-based system to automate smart home actions using geofencing.

## Project Goal

This project aims to detect a user's entry or exit from a predefined geographic area (a geofence) and trigger an action on a connected smart home device.

## Example Use Case

- When the user's phone enters the "Home" geofence, turn on the living room lights.
- When the user's phone exits the "Home" geofence, turn off all lights and lock the smart locks.
